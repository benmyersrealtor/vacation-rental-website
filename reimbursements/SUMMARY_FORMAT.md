# Trip Receipt Summary Format

Default summary format Ben prefers for reimbursement summary PDFs:

## Include
- trip name
- trip date
- receipt vendor
- receipt total
- receipt photo(s)

## Exclude
- mileage
- combined total
- reimbursed by
- notes
- browser print headers/footers
- top margin date/time
- bottom margin file information

## Output preference
- clean PDF style
- simple, uncluttered layout
- suitable to send directly via Telegram
