# Myers Realty Reimbursement Workflow

Default workflow for future Myers Realty reimbursements.

## Intake
- Ben sends receipts via Telegram.
- Confirm visible details first.
- Ask only for missing details needed to store and reimburse properly.

## Storage
- Save original receipt files under `reimbursements/YYYY/MM/`.
- Rename with consistent dated filenames.
- Log each reimbursement item in `reimbursements/tracker.csv`.

## Summary output
- Create a text-only accountant summary with:
  - itemized list
  - dates
  - vendors
  - amounts
  - total reimbursement requested

## Email draft
- Draft the reimbursement email to **Crystal** by default.
- Keep it casual, not stiff.
- Include the full itemized list and total.
- Mention receipts are available if needed.

## Goal
Make future Myers Realty reimbursement batches easy to recall, summarize, and send.
